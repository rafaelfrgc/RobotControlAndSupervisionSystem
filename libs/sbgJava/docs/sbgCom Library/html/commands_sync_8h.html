Set a logic input channel setting

Parameters
[in]handleA valid sbgCom library handle.
[in]channelChannel to configure. May Be 0 or 1
[in]inputTypeType of the logic input, may be
  • SBG_IN_DISABLED
  • SBG_IN_EVENT
  • SBG_IN_TIME_PULSE
    • SBG_IN_ODOMETER
      Parameters
      [in]sensitivitySensitivity of the trigger. It may be:
    • SBG_IN_FALLING_EDGE
    • SBG_IN_RISING_EDGE
    • SBG_IN_LEVEL_CHANGE
      Parameters
      [in]locationPhysical location of the input pin for the syncIN channel 0 on IG-500E.
  • SBG_IN_STD_LOCATION (default value, leave to this value if not used)
  • SBG_IN_EXT_LOCATION
    Parameters
    [in]nsDelayDelay to be added before the actual trigger is taken into account (in nano seconds) delays up to 2seconds are allowed:
    This delay is only used when the input is set to:
    • SBG_IN_EVENT
    • SBG_IN_MAIN_LOOP_START
  • SBG_IN_TIME_PULSE
    When used with the time pulse event, this delay can be negative, in order to simulate the time pulse propagation time
    Returns
    SBG_NO_ERROR if no error
