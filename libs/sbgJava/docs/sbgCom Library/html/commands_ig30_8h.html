#define SBG_FILTER_OPTION_ENABLE_ATTITUDE   (0x10)

Enable the attitude/navigation computation.
If disabled, the device can output calibrated data at 200 Hz.

#define SBG_GPS_DISABLE_SBAS   (0x00)

GPS option, disable SBAS and ranging

#define SBG_GPS_ENABLE_SBAS_DIFF_CORRECTIONS   (0x01)

GPS option, enable SBAS corrections

#define SBG_GPS_ENABLE_SBAS_RANGING   (0x02)

GPS option, enable ranging for SBAS

#define SBG_GPS_ALTITUDE_ABOVE_MSL   (0x04)

GPS option, When set the GPS outputs an altitude above Mean Sea Level. Otherwise, altitude is above Ellipsoid

SbgErrorCode sbgSetFilterAttitudeOptions ( SbgProtocolHandle  handle,
uint32  filterOptions 
)

Defines some options regarding the Kalman Filter.
It's possible, for example, to enable/disable the attitude computation or to enable/disable gyro-bias estimation.

Parameters
[in]handleValid sbgCom library handle.
[in]filterOptionsThe Kalman filter options mask.
Possible values are:
  • SBG_FILTER_OPTION_ENABLE_ATTITUDE
Returns
SBG_NO_ERROR if no error.

float  gyroAccelsSampling,

float  kalmanFreq 
)

Defines the sensors filter cut-off frequencies and the update rate for the Kalman Filter.
If you set a setting to 0.0, the value will remain unchanged.

Parameters
[in]handleValid sbgCom library handle.
[in]gyroAccelsSamplingReserved for backward compatibility. Leave to 0.
[in]cutoffGyroGyroscopes low-pass filter cut-off frequency in Hz.
[in]cutoffAccelAccelerometers low-pass filter cut-off frequency in Hz.
[in]cutoffMagnetoMagnetometers low-pass filter cut-off frequency in Hz.
[in]kalmanFreqThe Kalman filter refresh rate.
Returns
SBG_NO_ERROR if no error.
