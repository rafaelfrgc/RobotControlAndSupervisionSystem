SbgErrorCode sbgSetContinuousMode ( SbgProtocolHandle  handle,
SbgContOutputTypes  contMode,
uint8  divider 
)

Define continous mode options, enabled/disabled and output frames speed.

Parameters
[in]handleA valid sbgCom library handle.
[in]contModeDefines which continuous mode is set. Possible values are:
  • SBG_CONT_TRIGGER_MODE_DISABLE
  • SBG_CONTINUOUS_MODE_ENABLE
  • SBG_TRIGGERED_MODE_ENABLE
[in]dividerMain loop frequency divider applied on output.
For example if the device filter frequency has been set to 100Hz and we are using a divider of 4,
we will have data outputted at 100/4 = 25Hz.
Returns
SBG_NO_ERROR if no error.
