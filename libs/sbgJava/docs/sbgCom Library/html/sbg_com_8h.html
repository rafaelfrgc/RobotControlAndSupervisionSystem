SbgErrorCode sbgComInit ( const char *  deviceName,
uint32  baudRate,
SbgProtocolHandle *  pHandle 
)

Initialize the sbgCom library.
Open the COM port and try to get the default output mask and output mode from the device.

Parameters
[in]deviceNameCommunication port to open ("COM1" on Windows, "/dev/ttysX" on UNIX platforms).
[in]baudRateBaud rate used to communicate with the device.
Possible values are:
  • 9600
  • 19200
  • 38400
  • 57600
  • 115200
  • 230400
  • 460800
  • 921600
[out]pHandleUsed to returns the created SbgProtocolHandle struct.
Returns
SBG_NO_ERROR if a connection with the device has been established.
