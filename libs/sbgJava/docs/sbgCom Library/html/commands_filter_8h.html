Enumeration Type Documentation

enum _SbgHeadingSource

List of available sources for heading corrections.

Enumerator
SBG_HEADING_SOURCE_NONE 

The heading is calculated using gyroscopes only.

SBG_HEADING_SOURCE_MAGNETOMETERS 

The heading is based on magnetometers information and gyroscopes.

SBG_HEADING_SOURCE_GPS_COURSE 

GPS course is used

SBG_HEADING_GPS_ACCELERATIONS 

Heading is derived from GPS acceleration measurements

SBG_HEADING_SOURCE_USER 

Heading is fed by user via the main protocol

SBG_HEADING_SOURCE_REMOTE_MAG 

Remote magnetometers used as a heading source

SBG_HEADING_SOURCE_REMOTE_TRUE_HEADING 

Remote true heading sensor used (dual antenna)

Defines the kalman filter source for heading estimate.

Parameters
[in]handleValid sbgCom library handle.
[in]sourceSource used in heading calculation.
Possible values are:
  • SBG_HEADING_SOURCE_NONE, unavailable for IG-500N devices
  • SBG_HEADING_SOURCE_MAGNETOMETERS
  • SBG_HEADING_SOURCE_GPS, only for IG-30G and IG-500N devices
  • SBG_HEADING_SOURCE_USER
Returns
