enum _SbgAidingPosSrc

Define the position aiding source to use in the Kalman Naviagation filter.

Enumerator
SBG_POS_SRC_GPS 

We are using the GPS position as the aiding source for the navigation filter

SBG_POS_SRC_GPS_AND_BARO 

We are using the GPS position and the barometric altitude as the aiding source for the navigation filter

SBG_POS_SRC_USER 

We are using the user provided source for position as the aiding source for the navigation filter

enum _SbgAidingVelSrc

Define the velocity aiding source to use in the Kalman Naviagation filter.

Enumerator
SBG_VEL_SRC_GPS 

We are using the GPS velocity as the aiding source for the navigation filter

SBG_VEL_SRC_USER 

We are using the a user provided source for velocity as the aiding source for the navigation filter
