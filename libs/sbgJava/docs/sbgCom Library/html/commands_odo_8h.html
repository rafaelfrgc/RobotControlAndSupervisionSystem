Set the main configuration of the external odometer channels

Parameters
[in]handleA valid sbgCom library handle.
[in]channelOdometer Channel to use 0 or 1
[in]axisOdometer measurement axis (in sensor coordinate frame) May be:
  • SBG_ODO_X
  • SBG_ODO_Y
  • SBG_ODO_Z
[in]pulsesPerMeterdecimal number of pulses per meter
[in]gainErrorError in percent on the previous gain value
[in]gpsGainCorrectionIf set to TRUE, the GPS will automatically enhance the odometer gain
Returns
SBG_NO_ERROR if no error.
