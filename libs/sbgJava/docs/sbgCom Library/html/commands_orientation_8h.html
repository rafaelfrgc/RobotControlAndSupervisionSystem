SbgErrorCode sbgSetManualOrientationOffset ( SbgProtocolHandle  handle,
SbgOffsetType  offsetType,
const float  rotationMatrix[9] 
)

Defines the pre or post rotation to applied to the device.
This command is usefull to define an orientation 'offset'
You can either apply a pre rotation that applies directly on sensors values or
use a post rotation that only rotate attitude output such as euler angles.

The post rotation isn't available for the IG-500N because, sensors, attitude,
velocity and position should be expressed in the same frame.

Parameters
[in]handleValid sbgCom library handle.
[in]offsetTypeDefine if it's a pre or post rotation using:
  • SBG_OFFSET_PRE_ROT
  • SBG_OFFSET_POST_ROT
[in]rotationMatrix3x3 matrix that represents the rotation to apply.
Returns
SBG_NO_ERROR if no error.
