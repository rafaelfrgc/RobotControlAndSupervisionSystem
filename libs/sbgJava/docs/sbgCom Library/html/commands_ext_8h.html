Fast slew rate, baudrate is not limited. (default)

#define SBG_EXT_PORT_SLOW_SLEW   (0x0002)

Slow slew rate for EMI reduction. baudrate is limited to 230 400 bps.

SbgExtDeviceType  deviceType,
uint32  baudRate,
uint16  uartOptions 
)

Select the external device connected to the IG-500E and its UART configuration

Parameters
[in]handleA valid sbgCom library handle.
[in]deviceTypeThe device type connected
[in]baudRateThe baud rate used to communicate with the device at initialization
[in]uartOptionsSome uart Options (bitwize ORed) needed to communicate with the device. Restart the device if changed:
  • SBG_EXT_PORT_RS232 or SBG_EXT_PORT_RS422
  • SBG_EXT_PORT_DIS_EMI_REDUCTION or SBG_EXT_PORT_EN_EMI_REDUCTION
Returns
SBG_NO_ERROR if no error.
