SbgErrorCode sbgCalibMagnetometersSetTransformations ( SbgProtocolHandle  handle,
const float  offset[3],
const float  crossAxis[9] 
)

Define the current magnetometer calibration parameters.
This command is used, for example, by the sbgCenter when the magnetometer calibration is exectured on the computer.
When this command is called, the new settings are only stored in volatile memory.
To save this calibration permanently, use sbgCalibMagnetometers with SBG_CALIB_SAVE argument.

Parameters
[in]handleValid sbgCom library handle.
[in]offsetX,Y,Z offset of the magnetic field.
[in]crossAxis3x3 matrix containing the rotation and deformation of the magnetic field.
Returns
SBG_NO_ERROR if no error.
